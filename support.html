🔐 Ihre DSGVO-Rechte (Selbstbedienung)

Recht auf Auskunft (Art. 15 DSGVO)

Sie können jederzeit eine vollständige Kopie Ihrer bei uns gespeicherten Daten als JSON-Datei herunterladen:

Im Dashboard: Einstellungen → Datenschutz → Daten exportieren

Alternativ E-Mail an datenschutz@infinitymade.de — Bearbeitung innerhalb von 30 Tagen.

Recht auf Löschung (Art. 17 DSGVO)

Sie können Ihr Konto und alle damit verbundenen personenbezogenen Daten dauerhaft löschen:

Im Dashboard: Einstellungen → Datenschutz → Konto löschen

Achtung: Abrechnungsdaten bleiben aus gesetzlicher Aufbewahrungspflicht (§ 147 AO, § 304 SGB V) anonymisiert für 10 Jahre erhalten.

❓ Häufige Fragen

Wie kann ich mein Abo kündigen?

Dashboard → Einstellungen → Abo & Zahlung → Stripe Customer Portal → Abo kündigen. Kündigung greift zum Ende der laufenden Abrechnungsperiode.

Mein Google-Kalender synchronisiert nicht.

Kalender-Seite → Integration → Google neu verbinden. Falls das Problem bestehen bleibt: support@infinitymade.de mit Screenshot.

Wo finde ich meine Rechnungen?

Dashboard → Einstellungen → Abo & Zahlung → Stripe Portal → Rechnungshistorie.

Können Mitarbeiter sich selbst eintragen?

Ja. Im Dashboard erhalten Sie einen company_code — geben Sie diesen Code an Ihre Mitarbeiter weiter, sie registrieren sich auf /employee-signup.html.

📞 Eskalation

Kritischer Ausfall (Buchungen nicht möglich, Datenverlust)

Schreiben Sie eine E-Mail mit Betreff "DRINGEND" an support@infinitymade.de. Wir prüfen alle solchen Anfragen prioritär innerhalb 4h (werktags).
