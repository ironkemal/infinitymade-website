## Critical Anti-Patterns (do NOT repeat these)

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Service-role key in client | `service_role` ASLA browser'a iletilmez — sadece backend (`api-backend/`). Frontend `SUPABASE_ANON_KEY` kullanır. | blocking | RLS politikalarına güven; client-side `createClient(SUPABASE_ANON_KEY)` sabit. |
| Private key server'a sızdırma | PKCS#7 imzalamada `.p12` content / private key ASLA `POST` body'sinde gitmez. Yalnızca signed bytes + cert public metadata. | blocking | `runSignAbrechnung()` private key'i forge instance'ında tutar, sadece `signedBase64` + `certThumbprint/Subject/ValidTo/Serial` upload eder. |
| Heilmittel-Position default'a güvenip override UI eklememek | Default map (`defaultPositionForHeilmittel`) yaygın kodları kapsar ama özel durum (D1, KG-ZNS Kinder, telemed) hâlâ manuel pick ister. UI olmadan terapistler stuck. | advisory | Sprint 7-1 ilk öncelik. |
| ZAA error code sözlüğünü 40'ta bırakmak | Real-world DAS portal'ları farklı kod tanımları/varyantları kullanır. Eksik kod = "Unknown" görünür. | advisory | Pilot testlerde gelen her yeni kodu `error-translations.js`'e ekle. |
