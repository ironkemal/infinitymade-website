Schritt 3 / 7

Cal.com Verbindung

Hier verbindet der Assistent Ihren Online-Kalender. Buchungen werden automatisch eingetragen.

So erstellen Sie ein kostenloses Cal.com Konto

  1. Öffnen Sie cal.com/signup in einem neuen Tab.
  2. Registrieren Sie sich mit Ihrer geschäftlichen E-Mail.
  3. Wählen Sie einen Username (wird Teil Ihrer Booking-URL: cal.com/<username>).
  4. Bestätigen Sie die Bestätigungs-E-Mail.
  5. Im Cal.com Dashboard: Settings → Developer → API Keys → Add.
  6. Wichtig: Bei „Expires in" → Never wählen (sonst läuft der Bot nach 30 Tagen ab).
  7. Name: z.B. „InfinityMade Integration".
  8. Kopieren Sie den API-Key (beginnt mit cal_live_...) — er wird nur einmal angezeigt.
  9. Kommen Sie hierhin zurück und klicken Sie auf „Habe ich gemacht ↓".
